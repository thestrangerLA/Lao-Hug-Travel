# **App Name**: Voyager

## Core Features:

- Personalized Itinerary Generation: AI powered tool to create custom travel itineraries based on user preferences, budget, and interests, optimized for different travel styles (solo, group, family).
- Service Aggregation: Consolidate and display available tour packages, transport, guides, and lodging from a variety of providers.  Allow the user to filter and sort.
- User Authentication: Allow users to create accounts, and save their travel preferences and itinerary history.
- Social Sharing: Enable users to share their planned itineraries or travel experiences on social media platforms.

## Style Guidelines:

- Primary color: Teal (#26A69A) to evoke a sense of trust and exploration.
- Background color: Light Teal (#E0F2F1) - a lighter shade of teal to create a calm, inviting atmosphere.
- Accent color: Amber (#FFB300) - an analogous color to teal, providing contrast and warmth, for highlighting interactive elements.
- Headline Font: 'Playfair Display' (serif) for an elegant and classic feel.
- Body Font: 'PT Sans' (sans-serif) for readability and modern touch.
- Use consistent line-art icons, filled with the primary teal color, for service categories and user actions, providing a clear visual language.
- Implement a grid-based layout for a clean and structured presentation of information, ensuring easy navigation across devices.
